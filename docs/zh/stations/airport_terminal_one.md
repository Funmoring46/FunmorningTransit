# 机场1航楼站

![机场1航楼站出口A](https://github.com/Funmoring46/FunmorningTransit/blob/main/docs/assets/gallery/airport_terminal_one/stationentrance.png?raw=true){ width="750" }

机场1航楼站（英语：Airport Terminal 1 Station）是在山陆区域的机场捷运系统站。本站是让[山站](https://funmoring46.github.io/FunmorningTransit/zh/stations/mountains/)乘客方便去机场。本站在九月2025开通。

## 历史

Created on the creation of the Mountains Shanlu Airport it is one of the first stations to have a rail based APM in the server.
