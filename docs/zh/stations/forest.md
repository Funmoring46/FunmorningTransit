# 森林站

![森林站A出口](https://github.com/Funmoring46/FunmorningTransit/blob/main/docs/assets/gallery/forest/forest.png?raw=true){ width="750" }

森林站（英语：Forest Station）是在山陆区域的快线和村庄快线的换乘站。他是第二个先开的换乘站。（第一个是山站）

## 历史

从出口的竹子森林取名，本站是也是其中一个在2025八月底开的。
