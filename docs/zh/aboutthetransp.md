# 关于公共交通

在本服务器上个你可以使用公共交通去各种各样的地方但是有很多公司管理他们。 每个公司管理一个区域。

每个公司的简介：

**funmorning地铁**：管理1号线、2号线、机场捷运系统、家apm、快线、村庄快线、和3号线。

**兰纳铁路**：管理山-木山铁路、山-矿镇东铁路、和山-三地铁路。

**矿镇铁路**：管理矿镇-波斯顿铁路、1号线（矿镇）、和矿镇-矿镇东链接线。

**矿镇船**: 管理矿镇环和坛腾测试。

**花野铁路（仙台区）**：管理山-木山铁路扩展线、第二山-木山铁路扩展线、和轧道-坛腾铁路。

---

这个服务器有国家区域。这可以影响你的铁路旅程。

**山陆区**：你正在所在的地方。（山、重生点、机场⋯⋯）

**矿镇区**: 靠近火车站的村名。（矿镇、护照收集点、波斯顿⋯⋯）

**花野区**: 日本风格地区（木山、轧道⋯⋯）
