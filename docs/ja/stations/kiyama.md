# 木山駅

![木山駅A出口](../assets/gallery/kiyama/kiyama.png){ width="750" }

木山駅（英語：Kiyama Station；中国語：木山站）は、木山–山駅鉄道、木山–山駅鉄道延伸線、および第2木山–山駅鉄道延伸線の駅です。  
仙台エリアに位置し、2025年9月5日に開業しました。

## 歴史

開業当初、駅には出口も入口もなく、小規模な乗換駅として鍾乳洞（しょうにゅうどう／Shōnyūdō）方面への乗換と、木山–山駅鉄道延伸線への乗車待ちのみ行える状態でした。  
しかし、鍾乳洞方面の路線は駅と路線の距離が短すぎ、必要性が低かったため廃止されました。

2025年11月3日、木山駅は2つの駅に分割され、花野エリアへ入るためにはパスポートが必要となりました。

### 木山駅の分割

駅が分割された際、旧・木山–山駅鉄道のホームは木山検問所（木山検問所）として独立し、利用者はセキュリティチェックを通ってから木山駅へ入る仕組みになりました。

新しい木山駅の特徴：
- 出口が整備された  
- 2つの延伸線へ乗り換え可能  
- 駅施設は洞窟内に位置する
