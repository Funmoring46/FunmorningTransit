# レイクサイド・フォレスト駅

![レイクサイド・フォレスト駅](../assets/gallery/lakesideforest/lakesideforest.png){ width="750" }

レイクサイド・フォレスト駅（英語：Lakeside Forest Station；中国語：河边森林站）は、山路エリアに位置する1号線の駅です。  
2025年9月に開業し、湖のそばに森林があることからこの名前が付けられました。  
1号線では島式ホームのレイアウトが採用されています。

## 追加情報

レイクサイド・フォレスト駅の中国語名は、当初 **河**边森林站 と誤訳されましたが、正しくは **湖**边森林站 です。しかし、歴史的理由により元の名前がそのまま保持されています。
