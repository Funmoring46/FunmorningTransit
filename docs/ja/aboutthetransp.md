# 公共交通について

このサーバーでは、さまざまな公共交通を利用していろいろな場所へ移動できます。しかし、すべての交通は同じ会社が運営しているわけではありません。各会社がそれぞれの地域と路線を管理しています。

各社の紹介：

**funmorning メトロ**：  
1号線、2号線、空港 APM、ホーム APM、快速線、ビレッジ快速、そして 3号線を管理しています。

**ランナー鉄道**：  
山−木山鉄道、山−矿镇東鉄道、山−三地鉄道を管理しています。

**矿镇鉄道**：  
矿镇−ボストン鉄道、1号線（矿镇）、矿镇−矿镇東連絡線を管理しています。（ランナー鉄道の子会社）

**矿镇船務**：  
矿镇ループと タンテンテスト線 を管理しています。

**花野鉄道（仙台区）**：  
山−木山鉄道延伸線、第2山−木山鉄道延伸線、そして 轧道−タンテン鉄道を管理しています。

---

このサーバーには地域区分があり、これが鉄道の運行範囲や所属に影響します。

**山陸区**：  
あなたが現在いる可能性が高い地域です。（山岳地帯、スポーン地点、空港など）

**矿镇区**：  
駅の近くにある村の名前です。（矿镇、パスポート受取所、ボストンなど）

**花野区**：  
日本風のエリアです。（木山、轧道 など）
