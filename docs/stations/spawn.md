# Spawn Station

![Spawn Station Entrance A](../assets/gallery/spawn/spawn.png){ width="750" }

Spawn Station (Chinese: 重生点站) is a station on Line 2 and the future Ocean Rail Line. It is located in Shanlu area, and it opened in September 5, 2025.
