# Dripstone Caverns Station

![Dripstone Caverns Station](../assets/gallery/dripstone_caverns/dripstone_caverns.png){ width="750" }

Dripstone Caverns Station (Chinese: 滴水石洞穴站) is a station on the Line 1 located in the Kuangzhen area. It was opened around October 2025.
