# Tantem Cave Station

![Tantem Cave Station](../assets/gallery/tantem_cave/tantem_cave.png){ width="750" }

Tantem Cave Station (Chinese: 坛腾洞穴站) is a station on the Tantem Test in the Kuangzhen area. It is currently not completed as of January 11, 2026. The opening date is unknown.
