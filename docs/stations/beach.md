# Beach Station

![Entrance A of Beach Station](../assets/gallery/beach/beach.png){ width="750" }

Beach Station (Chinese: 沙滩站) is a station on Line 3 in the Shanlu area. It was opened around late October 2025.

## History

Named after all the beach around the area, it was created for the test of small stations in funmorning transit.
