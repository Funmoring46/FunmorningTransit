# Jungle Station

![Jungle Station](../assets/gallery/jungle/jungle.png){ width="750" }

Jungle Station (Chinese: 丛林站) is a station on Line 1 and the Kishimidō-Tantem Line in the Kuangzhen area.  It was opened around October 2025.

## History
Named after the jungle at the exit, it is the only station to have no immigration when going to Sendai. This will be changed soon. (1) 
{ .annotate }

1.  As of January 11, 2026 it has not changed yet.
