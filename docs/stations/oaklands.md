# Oaklands Station

![Entrance A of Oaklands Station](../assets/gallery/oaklands/oaklands.png){ width="750" }

Oaklands Station (Chinese: 橡木地站) is a station on Line 3 in the Shanlu area. It was opened around late October 2025.

## History

Named after all the oak forest around the area, it was created with Line 3 for testing.
