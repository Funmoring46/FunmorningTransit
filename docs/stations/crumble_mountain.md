# Crumble Mountain Station

![Exit A of Airport Terminal 1 Station](../assets/gallery/crumble_mountain/crumble_mountain.png){ width="750" }

Crumble Mountain Station (Chinese: 渣渣山站) is a station on Line 1 in the Shanlu area. It was opened around the start of September 2025.
